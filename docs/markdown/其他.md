# 进阶
此章节是基于vuepress的md文件的扩展语法

## 提示
**输入**
```md
type可选值: tip warning danger details
::: <type> [title]
[content]
:::
```
**输出**
::: tip 提示
这是一个提示
:::
::: warning 警告
这是一个警告
:::
::: danger 危险
这是一个危险提示
:::
::: details 详情
这里是详情
:::

## 显示隐藏内容
**输入**
```md
<details>
<summary>显示/隐藏内容</summary>
```vue
<template>
    <div>vue</div>
</template>
<script>
export default {};
</script>
```vue
</details>
```
**输出**
<details>
<summary>显示/隐藏内容</summary>

```vue
<template>
    <div>vue</div>
</template>
<script>
  export default {};
</script>
```
</details>

## Code Group
**输入**
```md
    :::: code-group
    ::: code-group-item FOO
    ```ts
    const foo = 'foo'
    ```
    :::
    ::: code-group-item BAR
    ```ts
    const bar = 'bar'
    ```
    :::
    ::::
```

**输出**
:::: code-group
::: code-group-item FOO
```ts
const foo = 'foo'
```
:::
::: code-group-item BAR
```ts
const bar = 'bar'
```
:::
::::
