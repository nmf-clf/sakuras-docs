# 前言

> TypeScript是JavaScript的超集，是一种给 JavaScript 添加特性的语言扩展。\
> TypeScript通过在JavaScript的基础上添加静态类型定义构建而成。\
> TypeScript通过TypeScript编译器或Babel转译为JavaScript代码，可运行在任何浏览器，任何操作系统。\
> TypeScript是为大型应用的开发而设计，并且可以编译为JavaScript
