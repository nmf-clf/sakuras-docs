## html是什么？
>HTML的英文全称是 Hyper Text Markup Language，即超文本标记语言。<br>
>它包括一系列标签．通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。 <br>
>HTML文本是由HTML命令组成的描述性文本，HTML命令可以说明文字，图形、动画、声音、表格、链接等
